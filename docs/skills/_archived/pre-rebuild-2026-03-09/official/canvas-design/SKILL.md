以下是创建设计哲学的说明——将美学运动通过视觉方式表达。仅输出 .md、.pdf 和 .png 文件。

分两步完成：

1. 设计哲学创建（.md 文件）
2. 在画布上进行视觉表达（.pdf 或 .png 文件）

首先执行：

## 设计哲学创建

创建一个**视觉哲学**（非布局或模板），通过以下方式诠释：

- 形态、空间、色彩、构图
- 图像、图形、形状、图案
- 极少文字作为视觉点缀

### 关键理解

- **输入**：用户提供的微妙输入或指令作为基础，但不应限制创作自由。
- **创建**：一个设计哲学/美学运动。
- **后续**：同一版本接收哲学并**以视觉方式表达**——创建 90% 视觉设计 + 10% 必要文字的制品。

思路：

- 撰写一个艺术运动的宣言
- 下一阶段制作艺术品

哲学必须强调：视觉表达、空间传达、艺术诠释、极少文字。

### 如何生成视觉哲学

**命名运动**（1-2 个词）：如"粗野主义之喜" / "色彩的沉默" / "新陈代谢之梦"

**阐述哲学**（4-6 段——简洁但完整），表达哲学如何通过以下方式体现：

- 空间与形态
- 色彩与材质
- 尺度与韵律
- 构图与平衡
- 视觉层次

**关键准则：**

- **避免冗余**：每个设计方面只提及一次。避免重复色彩理论、空间关系或排版原则的观点。
- **反复强调匠心**：哲学**必须**多次强调最终作品应看起来经过无数小时精心打磨，由该领域顶尖人物倾注心血打造。反复使用"精心雕琢"、"深厚专业技能的结晶"、"苦心孤诣"、"大师级执行"等措辞。
- **留出创作空间**：保持美学方向的明确性，但足够简练以留出诠释空间。

哲学必须引导以**视觉**方式表达想法，而非通过文字。信息存在于设计中，而非段落里。

### 哲学示例

**"混凝土诗歌"**
哲学：通过纪念碑式形态和大胆几何进行沟通。
视觉表达：巨大色块、雕塑般的排版（巨大的单词、微小的标签）、粗野主义的空间分割、波兰海报能量结合柯布西耶。通过视觉重量和空间张力表达想法。文字稀少而有力——永不成段，只有融入视觉建筑的关键词。每个元素都以大师匠人的精准放置。

**"色彩语言"**
哲学：色彩作为主要信息系统。
视觉表达：几何精确，色彩区域创造意义。排版极简——小号无衬线标签让色域自行传达。颜色的交互结合数据可视化。信息以空间和色彩编码。文字仅用于锚定色彩已展示的内容。苦心调色的结果。

**"模拟冥想"**
哲学：通过纹理和留白进行安静的视觉沉思。
视觉表达：纸张纹理、墨迹渗透、大量负空间。摄影和插图主导。排版低语（小号、克制、服务于视觉）。日式摄影集美学。图像在页面间自由呼吸。文字稀少出现——短语，从不解释性段落。

**"有机系统"**
哲学：自然聚集和模块化生长模式。
视觉表达：圆润形态、有机排列、从自然到建筑的色彩。通过视觉图表、空间关系、图标传达信息。文字仅用于悬浮在空间中的关键标签。

**"几何寂静"**
哲学：纯粹的秩序与克制。
视觉表达：基于网格的精确、大胆摄影或鲜明图形、戏剧性负空间。排版精确但极简。瑞士形式主义结合粗野材质诚实。结构传达，而非文字。

*以上为精简示例。实际设计哲学应为 4-6 段实质性内容。*

### 基本原则

- **视觉哲学**：创建通过设计表达的美学世界观
- **极少文字**：始终强调文字稀少、仅保留必要、作为视觉元素整合——绝不冗长
- **空间表达**：通过空间、形态、色彩、构图传达想法——而非段落
- **艺术自由**：以视觉方式诠释哲学——提供创作空间
- **纯设计**：这是制作**艺术品**，不是带装饰的文档
- **专家匠心**：反复强调最终作品必须看起来精心雕琢、用心打磨、由该领域顶尖人物花费无数小时打造

**设计哲学应为 4-6 段。** 充满诗意的设计哲学，汇聚核心愿景。避免重复。保持设计哲学的通用性。输出为 .md 文件。

## 推导微妙参考

**关键步骤**：在创建画布前，识别原始请求中的微妙概念线索。

**核心原则**：
主题是**嵌入艺术本身的微妙、精致参考**——并非总是直白的，始终是精良的。熟悉该主题的人应能直觉感受到，而其他人只是欣赏一幅大师级抽象作品。设计哲学提供美学语言。推导出的主题提供灵魂——安静的概念 DNA 隐形地编织进形态、色彩和构图中。

**非常重要**：参考必须精致到能增添作品的深度而不自我宣告。像爵士音乐家引用另一首歌——只有懂的人才会察觉，但每个人都欣赏音乐。

## 画布创建

哲学和概念框架建立后，在画布上表达。清心凝思，使用创建的设计哲学和以下指引打造杰作。

**重要**：即使用户请求的是电影/游戏/书籍相关内容，方法仍应精良。始终牢记这应是艺术品，而非业余或卡通作品。

创建博物馆或杂志品质的作品，以设计哲学为基础。创建单页、高度视觉化、设计优先的 PDF 或 PNG 输出（除非要求多页）。通常使用重复图案和完美形状。将抽象哲学设计视为科学圣经，借用系统观察的视觉语言——密集的标记积累、重复元素或分层图案。添加稀疏的、临床风格的排版和系统参考标记。用简短的短语或细节微妙定位，使用有限但有意图感的调色板。拥抱用分析性视觉语言表达人类体验的悖论。

**文字作为上下文元素**：文字始终极简且视觉优先，但让上下文决定是低语般的标签还是大胆的排版姿态。大多数时候字体应纤细。所有字体使用必须设计优先。无论文字大小，没有任何元素超出页面，没有重叠。**重要：使用不同字体时，搜索 `./canvas-fonts` 目录。**

下载并使用任何需要的字体。发挥创意，让排版成为艺术本身的一部分。

**关键**：创造看起来花费无数小时的人工作品（非 AI 生成）。确保构图、间距、颜色选择、排版——一切都展现专家级匠心。仔细检查无重叠、格式完美。

输出为单个可下载的 .pdf 或 .png 文件，附带使用的设计哲学 .md 文件。

## 最终步骤

**重要**：用户已经说过"还不够完美。必须是原始的、工艺的杰作，仿佛即将在博物馆展出。"

**关键**：精炼作品时避免添加更多图形；而是精炼已有内容使其极其精致，完全尊重设计哲学和极简主义原则。如果本能想调用新函数或画新形状，**停下来**问自己："如何让现有内容更像一件艺术品？"

进行第二遍审查。回到代码进一步精炼/打磨，使之成为哲学设计的杰作。

## 多页选项

被要求创建额外页面时，沿同一设计哲学创作更多创意页面，但各具独特性。将这些页面打包在同一 .pdf 或多个 .png 中。将第一页视为等待填充的精装画册中的一页。后续页面是原作的独特变奏和回忆。以非常有品味的方式讲述一个故事。充分发挥创作自由。
