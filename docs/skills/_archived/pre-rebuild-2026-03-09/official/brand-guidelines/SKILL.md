# 品牌视觉规范

## 概览

获取品牌官方身份和风格资源。

**关键词**：品牌、企业形象、视觉形象、后期处理、样式、品牌色彩、字体排版、品牌风格、视觉格式化、视觉设计

## 品牌指南

### 色彩

**主色调：**

- 深色：`#141413` - 主要文本和深色背景
- 浅色：`#faf9f5` - 浅色背景和深色上的文本
- 中灰：`#b0aea5` - 次要元素
- 浅灰：`#e8e6dc` - 微妙背景

**强调色：**

- 橙色：`#d97757` - 主要强调
- 蓝色：`#6a9bcc` - 次要强调
- 绿色：`#788c5d` - 第三强调

### 字体排版

- **标题**：Poppins（备选 Arial）
- **正文**：Lora（备选 Georgia）
- **注意**：建议预装字体以获得最佳效果

## 功能

### 智能字体应用

- 对标题（24pt 及以上）应用 Poppins 字体
- 对正文应用 Lora 字体
- 自定义字体不可用时自动回退到 Arial/Georgia
- 在所有系统上保持可读性

### 文本样式

- 标题（24pt+）：Poppins 字体
- 正文：Lora 字体
- 根据背景智能选择颜色
- 保留文本层次结构和格式

### 形状和强调色

- 非文本形状使用强调色
- 在橙色、蓝色和绿色强调色之间循环
- 保持视觉趣味性同时符合品牌规范

## 技术细节

### 字体管理

- 可用时使用系统安装的 Poppins 和 Lora 字体
- 自动回退到 Arial（标题）和 Georgia（正文）
- 无需安装字体——使用现有系统字体即可工作
- 为获得最佳效果，建议预装 Poppins 和 Lora 字体

### 色彩应用

- 使用 RGB 色值确保精确的品牌匹配
- 通过 python-pptx 的 RGBColor 类应用
- 在不同系统间保持颜色一致性
