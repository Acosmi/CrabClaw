此技能指导创建独特的、生产级的前端界面，避免千篇一律的"AI 流水线"审美。实现真正可运行的代码，极致关注美学细节和创意选择。

用户提供前端需求：要构建的组件、页面、应用或界面。可能包含用途、受众或技术约束的上下文。

## 设计思维

编码前，理解上下文并确定**大胆的**美学方向：

- **用途**：这个界面解决什么问题？谁在使用？
- **风格**：选择一个极端：极简主义、极繁主义、复古未来主义、有机/自然、奢华/精致、趣味/玩具感、杂志/编辑风、粗野主义/原始风、装饰艺术/几何风、柔和/粉彩、工业/实用风等。风格选择丰富多样。以此为灵感，设计出忠于美学方向的作品。
- **约束**：技术要求（框架、性能、无障碍性）。
- **差异化**：什么让它**令人难忘**？人们会记住的那一个亮点是什么？

**关键**：选择一个清晰的概念方向并精确执行。大胆的极繁主义和精致的极简主义都可以——关键在于意图性，而非强度。

然后实现可运行代码（HTML/CSS/JS、React、Vue 等）：

- 生产级且功能完善
- 视觉冲击力强，令人难忘
- 具有清晰美学视角的连贯性
- 每个细节都精心雕琢

## 前端美学指南

重点关注：

- **字体排版**：选择优美、独特、有趣的字体。避免 Arial、Inter 等通用字体；选择能提升前端美学的独特字体——出人意料的、有个性的字体选择。用一种独特的展示字体配一种精致的正文字体。
- **色彩与主题**：坚持一个连贯的美学。使用 CSS 变量保持一致性。主色调加鲜明点缀优于平均分布的保守配色。
- **动效**：使用动画创造效果和微交互。HTML 优先使用纯 CSS 方案。React 可用时使用 Motion 库。专注于高影响力时刻：一个精心编排的页面加载配合错峰展现（animation-delay）比分散的微交互更令人愉悦。使用令人惊喜的滚动触发和悬停状态。
- **空间构图**：出人意料的布局。不对称。叠加。对角线流动。打破网格的元素。大方的留白或可控的密度。
- **背景与视觉细节**：创造氛围和层次感，而非默认纯色。添加符合整体美学的上下文效果和纹理。应用创意形式如渐变网格、噪点纹理、几何图案、分层透明、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加。

**绝不**使用千篇一律的 AI 生成审美：过度使用的字体（Inter、Roboto、Arial、系统字体）、陈词滥调的配色方案（尤其是白色背景上的紫色渐变）、可预知的布局和组件模式、缺乏上下文个性的模板化设计。

创意解读，做出出乎意料的选择，让人感觉是真正为该上下文设计的。每个设计都应独一无二。在明暗主题、不同字体、不同美学之间变化。**绝不**在不同作品中趋同于常见选择（例如 Space Grotesk）。

**重要**：实现复杂度应与美学愿景匹配。极繁主义设计需要精心的代码、丰富的动画和效果。极简或精致设计需要克制、精准，以及对间距、字体和微妙细节的细心打磨。优雅来自于对愿景的良好执行。
