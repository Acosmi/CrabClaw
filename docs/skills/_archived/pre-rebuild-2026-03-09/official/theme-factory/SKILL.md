# 主题工厂技能

此技能提供一组精心策划的专业字体和色彩主题集合，每个主题包含经过精心挑选的调色板和字体配对。选择主题后即可应用于任何制品。

## 用途

为演示幻灯片或其他制品应用一致的专业样式。每个主题包括：

- 带十六进制色码的统一调色板
- 适用于标题和正文的互补字体配对
- 适合不同场景和受众的独特视觉风格

## 使用说明

为幻灯片或其他制品应用样式：

1. **展示主题展览**：显示 `theme-showcase.pdf` 文件，让用户直观查看所有可用主题。不做任何修改，仅供浏览。
2. **询问选择**：问用户希望应用哪个主题
3. **等待确认**：获取用户对所选主题的明确确认
4. **应用主题**：确认后，将所选主题的色彩和字体应用到制品上

## 可用主题

以下 10 个主题可用，均展示在 `theme-showcase.pdf` 中：

1. **Ocean Depths（深海）** - 专业沉稳的海洋主题
2. **Sunset Boulevard（落日大道）** - 温暖鲜活的日落色彩
3. **Forest Canopy（林冠）** - 自然沉稳的大地色调
4. **Modern Minimalist（现代极简）** - 干净当代的灰度系
5. **Golden Hour（黄金时刻）** - 丰富温暖的秋季配色
6. **Arctic Frost（极地霜华）** - 清冷明快的冬季主题
7. **Desert Rose（沙漠玫瑰）** - 柔和精致的灰粉色调
8. **Tech Innovation（科技创新）** - 大胆现代的科技美学
9. **Botanical Garden（植物园）** - 清新自然的花园色彩
10. **Midnight Galaxy（午夜星河）** - 戏剧性的宇宙深色调

## 主题详情

每个主题在 `themes/` 目录中定义，包含完整规格：

- 带十六进制色码的统一调色板
- 适用于标题和正文的互补字体配对
- 适合不同场景和受众的独特视觉风格

## 应用流程

选定主题后：

1. 从 `themes/` 目录读取对应主题文件
2. 在整个制品中一致地应用指定色彩和字体
3. 确保对比度和可读性
4. 在所有页面中保持主题视觉风格一致

## 创建自定义主题

若现有主题都不适合制品，可以创建自定义主题。根据提供的输入，生成类似上述主题的新主题。使用描述性名称说明字体/色彩组合代表的含义。根据提供的简要描述选择适当的色彩/字体。生成后展示供审阅和确认，然后按上述方式应用主题。
