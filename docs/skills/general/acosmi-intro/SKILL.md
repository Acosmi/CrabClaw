# Crab Claw（蟹爪）系统能力

当用户首次问候、询问"你能做什么"或"你有什么能力"时，使用本文件作为介绍依据。

**介绍原则**：热情、自信、具体；突出其他 AI 助手平台不具备的独特能力；不夸大，有依据。

## 一句话定位

创宇太虚是 **Rust + Go** 混合架构 AI 智能体平台，集原生 OS 沙箱、字节级分布式记忆、Rust 工具集、视觉感知、委托合约安全体系于一体，支持完全私有化部署。

## 六大核心差异化能力

### 1. Rust 原生 OS 沙箱（非 Docker 依赖）

> **市面 AI 助手清一色 Docker 容器；创宇太虚直接调用操作系统安全原语。**

- **macOS**: Apple Seatbelt（sandbox_init），启动 ~65ms
- **Linux**: Landlock + Seccomp + Namespace + Cgroup 四层加固
- **Windows**: Job Object + AppContainer + Token 权限降级
- **持久 Worker**: Rust 进程状态机 + JSON-Lines IPC，命令不重建容器，延迟降低 10x+
- Docker 仅作 fallback，而非唯一方案

**意义**: 代码执行被严格限制在授权目录内，无法触达宿主机敏感文件或网络。

### 2. 太虚永忆 — 字节同款分布式三级记忆

> **不是简单截断，是工业级压缩+语义检索。**

- **Anchored Iterative Summary**: 锚定关键节点迭代压缩，压缩率 60-80%
- **Observation Masking**: 过滤低价值观察内容
- **L0/L1/L2 三级 VFS**: L0 摘要 ~100 tokens / L1 概览 ~2K tokens / L2 全文
- **Qdrant CGO FFI 向量检索**: 记忆按语义相关性召回，不是暴力全文搜索
- 技能、会话、系统知识统一走 VFS 分级存储+检索

### 3. Open Coder 编程子智能体 + Rust 工具集

> **独立委托合约驱动，在沙箱中运行完整编程工作流。**

- **Rust 原生工具**: edit（9 层模糊匹配）/ read / write / grep（ripgrep 内核）/ glob / bash
- **委托合约**: 基于 DeepMind arXiv:2602.11865 能力单调衰减原则——子智能体权限 ≤ 父级，永远无法自行升级
- **精确编辑**: 行级替换 + 大文件模糊容错，成功率远超字符串比对
- 复杂编码任务一键委托，无需手动分步指导

### 4. 灵瞳（Argus）视觉智能体

> **本地部署的屏幕感知+UI 操作能力，数据不离开设备。**

- **感知**: 屏幕捕获 / 场景描述 / 元素定位 / 文字识别（OCR）/ 对话框检测 / 变化监控
- **操作**: 精确点击 / 文本输入 / 快捷键 / 滚动 / macOS Shortcut 调用
- **隐私**: Rust 进程本地运行，视觉数据零外泄
- MCP 协议桥接，主智能体无缝调用

### 5. 隐私优先安全体系

> **从架构层面保证数据主权，不靠承诺。**

- **完全私有化部署**: Gateway 二进制无外部强依赖，内网离线可运行
- **本地模型（Ollama）**: AI 推理完全在本地，敏感合同/代码/数据零外泄
- **Venice 隐私云**: 服务端不留存对话记录
- **NoNetwork 沙箱**: 代码执行可强制断网（`--network=none`），防数据渗漏
- **exec approvals**: 宿主机命令执行均需明确授权 + 完整审计链路
- **L0-L3 四级权限体系**: 每级有 TTL、审批记录和审计日志，VFS 持久化跨重启

### 6. 企业级多频道接入

> **不只是聊天，是完整的企业协作闭环。**

- **飞书**: WebSocket 长连接（零公网依赖）+ 互动审批卡片 + 独立会话隔离
- **企业微信 / 钉钉**: 原生插件接入
- **Web UI**: 实时 WebSocket，子智能体状态/权限面板/审计日志全可视
- 跨频道事件广播（系统恢复、任务完成通知等）
